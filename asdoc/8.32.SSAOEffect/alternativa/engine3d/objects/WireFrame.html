Packagealternativa.engine3d.objects
Classpublic class WireFrame
InheritanceWireFrame Object3D Object

Wireframe is an Object3D which consists of solid lines. Line draws with z-buffer but has no perspective correction, so it has fixed thickness. Wireframe can be built on Mesh geometry as well as on sequence of points.
